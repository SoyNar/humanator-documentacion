El proyecto **Humanator** se desarrollará utilizando las siguientes tecnologías:

### Backend
- **PHP 8.1 o superior**
- **Laravel 10/11** como framework principal.
- **Scrumble** para documentar automáticamente la API REST.
- Patrones de diseño como **DDD (Domain-Driven Design)** y arquitectura limpia.
- Documentación de código con **PHPDoc**.

### Frontend
- **Angular 16 o superior**
- Uso de **módulos**, **signals** e **inputs** (no standalone components)
- **PrimeNG** para componentes UI.
- Documentación de componentes con **TypeDoc** o **JSDoc**.

### Base de datos
- **MySQL 8.x o postgresSql **
- Uso de migraciones y seeders de Laravel para la gestión de datos.
- Integridad referencial con claves foráneas y restricciones.

### Herramientas y documentación
- **Docusaurus con TypeScript** para centralizar la documentación del proyecto.
- Diagramas con **Mermaid** o **PlantUML** para ilustrar arquitectura y flujos de datos.
- Control de versiones con **Git**.

### Otros
- **Composer** para la gestión de dependencias PHP.
- **Node.js 18.x o superior** para Angular y herramientas frontend.
- Buenas prácticas de desarrollo, incluyendo **tests unitarios y de integración**.
