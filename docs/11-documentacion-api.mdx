# Documentación de Humanator

Esta página describe **cómo se documentará la aplicación Humanator**, incluyendo backend, frontend y herramientas de documentación.

## 1. API REST

- La API REST será documentada utilizando **Scrumble**, compatible con **Laravel 10 y 11**.
- Generará documentación **automática y clara** de los endpoints, parámetros, respuestas y errores.
- La documentación se mantendrá actualizada conforme se modifiquen o agreguen endpoints.
- Los endpoints seguirán buenas prácticas REST y convenciones de naming.

### Requerimientos de Scrumble

- **PHP 8.1 o superior**  
- **Laravel 10.x o superior**

### Instalación de Scrumble

Para instalar Scrumble en el proyecto Laravel, ejecuta:

```bash
composer require scramble/scrumble
